## [未发布]

### 新增

- **基准配置背板回归普通配件**：背板不再走 `bp_tri_pn`/`bp_dc_pn` 特殊字段，改为在底盘件清单里像其他料一样添加（选「背板」分类 + 三模/直连料号）；现有 4 个配置的背板已从字段迁回 parts 清单。配置页三模/直连便捷切换由 L6ChassisConfig 支持（无需改）
- **商机驾驶舱时间维度自定义区间**：三快捷按钮（本周/本月/本年）旁加「自定义」浮层，支持上周 / 上月 / 去年 / 近30天 / 近90天 / 指定月份 / 任意日期区间；后端 `/api/dashboard/summary` 新增 `start` / `end` 参数
- **商机列表按更新时间排序**：筛选区加升降序下拉，后端 `/api/opportunities/list` 新增 `sort_by` / `sort_order`

### 优化

- **驾驶舱图表粒度自适应区间**：≤10 天按天、11–90 天按周、>90 天或跨年按月（月维度区间不再按天堆 30 个点）

### 重构

- **料号库编辑页按类别分区**：基础信息 / 规格参数 / 扩展属性三段；规格参数按类别动态渲染专用字段（电源功率、CPU/GPU/散热器 TDP、机箱/背板盘位、线缆分组、Riser/OCP 槽位），杜绝同字段两处编辑
- **服务器-管理面 4 组件接入玻璃视觉系统**：料号库 / 机型 / BOM模板 / 基准配置的实色 `.panel` 迁至 `.glass` / `.glass-light`，与商机驾驶舱统一视觉，消除 dark-first 遗留
- **报价工作台 KP / L6 卡片布局与玻璃统一**：
  - 布局：L6 利润率条移到机箱卡上方、Key Parts 合计移到 KP 区底部 → 两卡对称（利率调整在上、总价在下）
  - 玻璃：Excel 模式 KP 卡从 flat `.glass-light` 单色改为机型目录页 `.sc-type-card` 同款配方（`glass-1-bg→glass-2-bg` 双层渐变 + `shadow-md` + inset 高光 + hover 蓝边环）；同时摘掉 KP 区外层 `.card-kp` 的 `.glass` 外套，消除"玻璃套玻璃"——内层 backdrop-filter 模糊到外层已模糊的白雾、两层 blur 叠成死灰糊状。KP 卡现为单层玻璃直坐页面背景，与机型目录页观感一致（新建模式 `.sc-panel` 卡同样受益于脱层）

### 修复

- **L6 价格历史快照丢失**：`l6_repo.py` 的 `save_history_snapshot` / `get_history` 原生 SQL 写 `l6.l6_price_history`（表实际不存在，与 model 声明的 `l6_history` schema 不符），导致 2026-07-04 后价格快照 save 一直失败、历史全丢；统一 SQL 到 `l6_history.l6_price_history`
- **迁移后序列未同步（系统性遗留）**：迁移插带 id 数据后各表 SERIAL 序列停在 1，新 INSERT 撞主键。新增 `scripts/sync_sequences.py` 批量 `setval` 全部 34 个序列到 max+1（幂等）
- `PartsMasterRepository` 补 `update()` 方法（原缺失导致料号编辑保存 500）；类别输入由 `a-input` 改 `a-auto-complete`

### 清理

- 删除料号 `specs` 与顶层列的冗余字段（`kind` / `description` / `note`，迁移副产物），一次性幂等脚本
- 删除前端「供应商」死表单项（表无此列，一直被静默丢弃）

### 文档

- 重写 `Frontend_Style_Guide.md`：反映当前 Soft Glassmorphism + 双主题现状，取代过时的 Cyberpunk Dark 版
- `Frontend_Style_Guide.md` §3 新增「避免玻璃层叠」：玻璃卡是唯一玻璃层、直坐页面背景；需要分组用透明布局容器，别再套 `.glass`（嵌套会让内层 backdrop-filter 模糊到外层白雾、整体发灰）。同步加 §10 Don't 与 §12 FAQ
- `Principles.md` 新增第⑩条「视觉规范同步」：改 token 必须同步风格指南

## [0.1.0] - 2026-06-29

### 版本概述
从旧系统 0.2.5（Streamlit 单体）迁移至 Vue3 + FastAPI 前后端分离架构。

### 新增
- 智能上传（Excel 拖拽 → 解析 → 多配置拆分）；报价工作台（实时精算 / L6+KP 独立调价 / 财务联动）；项目管理（看板 / 校对 / 导出）；基准价格（KP/L6 在线 CRUD + 趋势）；核心引擎（五维匹配 / 质保按需 / WYSIWYG 导出）

### 技术栈
- 前端：Vue3 + TypeScript + Vite + Ant Design Vue + Pinia
- 后端：FastAPI + SQLAlchemy
- 数据库：SQLite × 4 独立库
- 样式：GitHub Dark 暗色主题（CSS 变量系统）
